# Resume Builder Simple

A simple web app to build and preview a resume directly in the browser.

## Project Structure

- `index.html` - main page and layout
- `src/styles.css` - UI styling
- `src/main.js` - interactive behavior (form to preview)
- `data/sample-resume.json` - sample data for quick testing

## Run Locally

1. Clone the repository.
2. Open `index.html` in your browser.

No build tools required for this starter version.
